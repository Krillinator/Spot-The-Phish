# Spot the Phish

**Spot the Phish** is a browser-based phishing awareness game.  
Users analyze realistic examples and decide whether a message is **phishing** or **legitimate**, then receive immediate feedback explaining why.

The project is fully static and runs entirely in the browser.

---

## Live demo

https://krillinator.github.io/Spot-The-Phish/

---

## How it works

1. Start the game from the landing page  
2. Review a message or scenario  
3. Decide if it is phishing or not  
4. Receive clear feedback on what to look for

## Purpose

The goal is to improve phishing awareness by focusing on:
- Subtle warning signs
- Context and intent
- Common attacker techniques

Designed for education and self-paced learning.

---

## Tech stack

- HTML
- CSS
- JavaScript (ES modules)
- GitHub Pages

No frameworks. No backend. No build step.
